# Sistema 5 - Plataforma de Gestión de Activos Digitales (DAM) para Medios de

Se requiere implementar una plataforma de Gestión de Activos Digitales (DAM) para un grupo de medios de comunicación que maneja millones de archivos de video, audio e imágenes. El sistema debe permitir el almacenamiento a largo plazo, la indexación inteligente (mediante metadatos y análisis de contenido con IA), la búsqueda rápida por parte de los editores, y la distribución automatizada de contenido a múltiples canales (web, TV, redes sociales). La velocidad de recuperación de archivos grandes es esencial, y el sistema debe ser escalable para manejar un volumen de datos en constante crecimiento.
